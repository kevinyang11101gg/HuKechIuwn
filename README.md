# 项目简介
## 核心功能
- 数据清洗：包括缺失值处理和去重处理
- 数据预处理：提取有用信息
- 数据统计：对数据进行分析和统计
- 数据可视化：使用matplotlib进行数据可视化展示
## 适用场景
- 美妆行业市场分析
- 销售趋势预测
- 产品效果评估
## 技术特点
- 采用Python进行数据分析与可视化
- 使用pandas和numpy进行数据处理
- 利用matplotlib进行数据可视化展示
